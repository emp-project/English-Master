# Contributing to English Master

Thank you for your interest in contributing to English Master.

## How to contribute

- Keep the database structure unchanged.
- Follow the project naming conventions.
- Check data before committing.
- Write clear commit messages.

## Commit message examples

feat(database): add TOEIC vocabulary
fix(database): correct IPA
docs: update README

## Pull Request

Please describe:
- What was added?
- What was changed?
- Why was it changed?

Thank you for helping improve English Master.
