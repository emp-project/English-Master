# 🚀 English Master

> An open English learning platform for TOEIC, IELTS, VSTEP and General English.

## Vision

English Master is an open-source project designed to help learners build English vocabulary and language skills through a structured and reusable knowledge base.

Instead of creating separate materials for TOEIC, IELTS or VSTEP, English Master provides a unified database that supports multiple learning paths.

## Project Goals

- 📚 Build a high-quality English knowledge database
- 🎯 Support TOEIC, IELTS, VSTEP and General English
- 💻 Desktop (Excel)
- 🌐 Website
- 📱 Mobile App
- 🤖 AI-ready architecture
